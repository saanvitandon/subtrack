# SubTrack
A Subscription Management Dashboard to track, manage, and analyze your recurring payments in one place.

# Overview

SubTrack is a web-based subscription management system designed to help users keep track of their recurring expenses such as Netflix, Spotify, Gym memberships, and more.

It allows users to:
- Sign up / Log in
- Add new subscriptions
- View all active subscriptions
- Analyze spending through an analytics dashboard

The goal of this project is to provide a simple yet effective tool to manage subscriptions and avoid unnecessary spending.

# Features

- User Authentication (Signup & Login)
- Add New Subscriptions
- Dashboard to View All Subscriptions
- Analytics Page to Track Spending
- Data stored using Local Storage
- Clean and Responsive UI

# Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage (for data persistence)
